### Datathon - Anime Dataset Statistical Analysis
**File:** datathon.py
 
A comprehensive statistical analysis and regression project on an **Anime dataset**, exploring what factors influence anime scores and popularity.
 
**Key Steps:**
- Cleaned and preprocessed the dataset (type conversion, null handling, deduplication)
- Descriptive statistics with range, mean and median comparisons
- Score distribution histogram and genre-based boxplots
- Outlier detection and removal using **IQR method**
- **Q-Q plots** for normality testing
- Correlation heatmap across numerical features
- **95% Confidence Interval** for mean score
- Margin of error at 90%, 95% and 99% confidence levels
- **Independent T-Test** - Action vs. Romance genres (hypothesis testing)
- **Pearson Correlation** - members vs. score
- **Linear Regression** - predicting score from member count (R² evaluation)
 
**Tools & Libraries:** pandas, numpy, matplotlib, seaborn, scipy.stats, scikit-learn, statsmodels
